## 环境依赖

- Node.js：v18 及以上版本
- pnpm.15.5 及以上版本
- 浏览器：Chrome、Firefox、Safari 等现代浏览器

## 安装步骤

1. 克隆项目代码到本地

```bash
git clone https://github.com/hankliu62/questions.git
```

2. 进入项目目录

```bash
cd questions
```

3. 安装项目依赖

```bash
pnpm install
```

4. 启动项目

```bash
pnpm run dev
```

## 使用方法

访问 `http://localhost:3000` 即可看到前端面试题的首页。

在首页上可以选择需要的工具进行使用。

工具的具体使用方法和参数可以参考每个工具的文档说明。
